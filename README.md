## 프로젝트 소개
- 클래스101에서 아이디어를 얻어 만들었습니다.

## 개발 기간
- 23.01.16 - 23.01.20

## 개발 환경
- `Java 11`
- **Framework** : `Spring boot(2.7.7)`
- **Database** : `H2 Database`
- **ORM** : `MySQL`

## 주요 기능
1. 회원가입
2. 로그인
3. 로그아웃

1. 프로필 작성
2. 프로필 조회
3. 강의 목록 조회

5. 튜터 정보 조회
6. 수강 신청
7. 튜터 권한 요청

2. 프로필 조회
3. 강의 등록
4. 강의 목록 조회
